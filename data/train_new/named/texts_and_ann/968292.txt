Собчак новым стихом завершила «и срач, и драку» со Шнуровым Телеведущая Ксения Собчак посвятила лидеру группы «Ленинград» Сергею Шнурову очередное стихотворение, в котором объявила об окончании «срача и драки» с артистом и призвала «погладить шнурки и идти на сцену». «Дуэль, похоже, состоялась. // Пусть не в эфире, а в сети, // И я, конечно, оказалась // Такая «господи-прости», // И «хитрожопая», и «дура». // Говно метал умело Шнуров, // Сам в белом фраке и пальто. // За что ты так, Сергей? За что?!! // Мне нужен рейтинг? Здесь, Сережа, // Ты с кем-то путаешь, похоже, // Безосновательно меня, // Меня при этом и говня. // Да, твой сарказм про телевизор // Явился для меня сюрпризом: // «Не твой говеный водоём»? // А как мы плавали вдвоем.... // В том водоеме... – позабыто? // Ты у разбитого корыта // Меня шпыняешь «Домом-2», // Обидные найдя слова? // За что, Сергей, за что мне это? // Ведь я же «Шнуров вокруг Света» // Не поминаю что ни день // Как ерунду и ***. // Ведь я не бью ни в глаз, ни в бровь // Словами: «Ваша «Про любовь» – // Пошлятина на постном масле, // Чьи рейтинги давно угасли», // ... Ведь то, что было, то прошло, // А злая память – западло! // По повеленью желтой моды // Ты нынче пишешь про разводы, // Хотя бы мог писать хиты... // Какой-то это, ***, не ты! // Погладь шнурки, иди на сцену, // Себе и песням вспомни цену, // Пока не близок простатит, // И Ошо сам тебя простит! // А также, может быть, и я, // Ведь мы же бывшие друзья, // А может, будущие – тоже. // Пока! Подумайте, Сережа», – написала Собчак на своей странице в Instagram. «P. S. На этом кончаю и срач, и драку. // Такой вот шоу-бизнес, мазафака!» – добавила она. Конфликт Собчак и Шнурова в Сети начался после того, как артист высмеял развод телеведущей с Максимом Виторганом. В ответ Собчак в стихотворной форме вызвала на баттл Шнурова, однако музыкант отказался, указав, что телеведущая пытается привлечь внимание за счет скандала. Впрочем, на этом словесная дуэль не закончилась. Собчак вновь в стихах ответила на отказ Шнурова участвовать в баттле, упрекнув его в трусости. Певец в ответ посвятил Собчак еще одно стихотворение. «Как личность ты, конечно, многогранна. // Но чем ты занята? Да *** (черт) тебя пойми. // Одно конкретно – ты мать сына Виторгана, // Все остальное так невнятно, мон ами. // Как бы ведущая, как бы политик, // Как бы жена, как бы подруга, // Как бы умна и как бы аналитик, // Как бы квадрат из как бы круга. // И все, что производишь ты на свете, // Лишь шум и водородная вода. // ... Стишки простые написать не в силах, // Ты их присвоила прекрасно хитрожопо. // Это, конечно, тоже очень мило. //Не спровоцирует всемирного потопа. // Живи как знаешь, ври всем и юли ты, // Пусть у тебя не будет целлюлита», – написал Шнуров. Именно в ответ на этот пост Собчак объявила об окончании «срача и драки».
