Шнуров объявил о последнем турне «Ленинграда» Лидер группы «Ленинград» Сергей Шнуров объявил о последнем турне музыкального коллектива. Артист опубликовал пост в Instagram, в котором отметил, что 20 лет без выходных сочинял «стишки и песни», а теперь задумался, не пора ли завершить деятельность группы. «Прыгать и скакать приматом, // Доложу, нелегкий крест. // Я за всех ругался матом. // Бог не сдаст, свинья не съест. // Между сложными вещами, // Я скажу, как не юнец, – // Этот тур будет прощальным, // Вот и сказочке конец. // Может быть, когда-нибудь, // Но нескоро. На миру, // Если в край не *** (достанут) // Ленинград я соберу. // А пока, ну все, пока. // Мавр сделал свое дело, – // Знаю я наверняка», – написал артист. Группа «Ленинград» была основана в 1997 году в Санкт-Петербурге. Лидером и автором большинства ее композиций стал Сергей Шнуров.
